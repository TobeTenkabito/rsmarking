Tile Service 全链路结构
前端地图（Mapbox / OpenLayers / Leaflet）
        │
        │  GET /tile/{index_id}/{z}/{x}/{y}.png?bands=1,2,3
        ▼
FastAPI 路由层
        │
        ├─ L1/L2 Cache（index_id + z/x/y + bands）
        │
        ├─ DB 查询
        │    raster_metadata
        │    └─ index_id → file_path
        │
        ├─ 文件系统
        │    file_path (GeoTIFF)
        │
        ├─ TileEngine
        │    ├─ CRS 校验 / 重投影
        │    ├─ tile → bounds
        │    ├─ raster window
        │    ├─ resampling
        │    ├─ band 处理
        │    └─ RGBA
        │
